<task type="checkpoint:human-action" gate="blocking">
  <name>Task 1: Create YouTube channel and configure branding</name>
  <read_first>.planning/ROADMAP-promotion.md (YouTube Channel Spec section)</read_first>
  <action>
Create the YouTube channel manually in a browser. Follow these exact steps:

1. Go to https://studio.youtube.com and sign in with Google account
2. Create a new channel with name: `hamtools`
3. Set handle: `@hamtools` (if taken, use `@hamtools-cc`)
4. Set channel description (About section):
   ```
   Open-source browser tools for amateur radio operators. Built by 2E0MIK.
   hamtools.cc
   ```
5. Set category: Science & Technology
6. Set language: English (UK)

Configure branding:
- Profile icon: stylized "HT" monogram or simple antenna icon, dark background (#0d1117)
- Channel banner: dark background (#0d1117) with `hamtools.cc` wordmark in white, 2560x1440px canvas (verify dimensions in YouTube Studio > Customization > Branding)

Configure default upload settings (YouTube Studio > Settings > Upload defaults):
- Category: Science & Technology
- Tags: `ham radio`, `amateur radio`, `CAT control`, `yaesu`
- License: Creative Commons Attribution

Create playlist:
- Name: `CAT FTX-1 Controller`
- Description: `Demos and tutorials for the CAT FTX-1 browser-based controller`
- Visibility: Public
  </action>
  <how-to-verify>
    1. Visit the channel URL (youtube.com/@hamtools or youtube.com/@hamtools-cc)
    2. Confirm channel name shows "hamtools"
    3. Confirm About section has description text
    4. Confirm profile icon and banner are set
    5. Confirm "CAT FTX-1 Controller" playlist exists (may be empty)
    6. Go to YouTube Studio > Settings > Upload defaults and confirm category/tags/license
  </how-to-verify>
  <acceptance_criteria>
    - Channel is publicly accessible at a youtube.com/@hamtools or youtube.com/@hamtools-cc URL
    - Channel description contains "Open-source browser tools for amateur radio operators"
    - Profile icon is set (not the default letter avatar)
    - Banner image is set (not blank)
    - Playlist "CAT FTX-1 Controller" exists
    - Default upload category is "Science & Technology"
    - Default license is "Creative Commons Attribution"
  </acceptance_criteria>
  <resume-signal>Paste the channel URL (e.g., youtube.com/@hamtools) to confirm completion</resume-signal>
  <done>YouTube channel is live, branded, and configured with defaults and playlist</done>
