  <action>
Record a 60-90 second screen recording showing the app in use. Setup and recording steps:

**Pre-recording setup:**
1. Open Chrome at https://cat-ftx1.hamtools.cc
2. Set browser window to approximately 1400px wide (the OS DPI is 125%, so this produces clean 1080p output)
3. Connect the FTX-1 via USB cable and power it on
4. Open Xbox Game Bar (Win+G) and verify capture is ready
   - If Game Bar refuses to record Chrome ("this app can't be captured"), install OBS Studio from obsproject.com and use Window Capture instead

**Recording sequence (target 60-90 seconds total):**
1. Show the app disconnected (dashboard visible, dimmed controls) — 3-5 sec
2. Click Connect, select serial port, show connection establishing — 5-10 sec
3. Show live VFO frequency updating in real time as you tune the radio knob — 10-15 sec
4. Execute a preset button (e.g., a frequency preset) — show the radio responding — 10-15 sec
5. Change VFO frequency via the app (click frequency digits or use scroll wheel) — 10-15 sec
6. Show band/mode change or squelch adjustment — 10 sec
7. Brief pause on the full UI to show all controls in connected state — 5 sec

**Export settings:**
- Resolution: 1920x1080 (1080p)
- Frame rate: 30 or 60 fps
- Format: MP4 (H.264)

**If using Game Bar:** Recording saves to Videos/Captures folder automatically at native resolution. Trim in Windows Photos app if needed.
**If using OBS:** Set output to 1920x1080, CRF 18-23, record to MP4.
