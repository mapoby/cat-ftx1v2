<task type="auto">
  <name>Task 2: Wire serial disconnect event (ERR-03)</name>
  <files>composables/useSerial.ts</files>
  <action>
Add three new private functions and wire them into `connect()` and `disconnect()`.

**1. Add `_handlePhysicalDisconnect` function** — place it after the `_drainQueue` function (around line 270):

```ts
function _handlePhysicalDisconnect(event: Event): void {
  if ((event as any).target !== _port) return
  _drainQueue(new Error('Serial port disconnected'))
  if (_reader) { try { _reader.cancel() } catch { } _reader = null }
  if (_writer) { try { _writer.releaseLock() } catch { } _writer = null }
  _port = null
  state.value = defaultState()
}
```

**2. Add `_wireSerialEvents` and `_unwireSerialEvents` functions** — place them immediately after `_handlePhysicalDisconnect`:

```ts
function _wireSerialEvents(): void {
  ;(navigator as any).serial.addEventListener('disconnect', _handlePhysicalDisconnect)
}

function _unwireSerialEvents(): void {
  ;(navigator as any).serial.removeEventListener('disconnect', _handlePhysicalDisconnect)
}
```

**3. Call `_wireSerialEvents()` in `connect()`** — add it immediately after the `_patch({ connected: true, ... })` line (currently line ~344), before the closing `}` of the try block:

```ts
_patch({ connected: true, baudRate, port: 'Web Serial', error: null })
_wireSerialEvents()
```

**4. Call `_unwireSerialEvents()` in `disconnect()`** — add it as the first line of the `disconnect()` function body, before the `_write('AI0')` best-effort call. This ensures the listener is removed whether or not subsequent steps throw:

```ts
export async function disconnect(): Promise<void> {
  _unwireSerialEvents()
  try { await _write('AI0') } catch { /* best-effort */ }
  // ... rest unchanged
}
```

The `_handlePhysicalDisconnect` guard `if ((event as any).target !== _port) return` ensures the handler only acts on the port we opened, not on other USB serial devices the user may have.

Do not add a guard for `isSupported` inside these functions — they are only ever called after `connect()` succeeds, which already requires `isSupported`.
